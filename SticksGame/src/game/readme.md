# Гра "Палички" (Консольна версія)

Це консольна реалізація класичної гри "Палички".

## Правила гри

1.  Початок: Гра починається з певної кількості паличок (N), яку задають гравці.
2.  Максимальний хід: Гравці також домовляються про максимальну кількість паличок (M), яку можна взяти за один хід.
3.  Процес гри: Два гравці ходять по черзі.
4.  Хід: У свій хід гравець повинен взяти будь-яку кількість паличок (K) за умови, що:
    * `1 <= K <= M`
    * `K` не може бути більшим, ніж кількість паличок, що залишилися.
5.  Визначення переможця: Програє той гравець, який змушений взяти **останню** паличку. Відповідно, інший гравець перемагає.

## Як запустити

1.  Відкрийте проект у IntelliJ IDEA.
2.  Знайдіть та запустіть файл `Main.java`.
3.  Дотримуйтесь інструкцій у консолі, щоб ввести початкову кількість паличок та максимальну кількість для ходу.
4.  Грайте!
